# Contributing to Chakraview

We welcome contributions from the security community to enhance the strategic depth of Chakraview.

### 🛡️ How to add a new Chakra (Module)
1. **Fork** the repository to your own account.
2. **Develop** a new reconnaissance module in the `modules/` directory.
3. **Standards**: Ensure the module includes error handling and follows the mythological naming convention.
4. **Initialization**: Your module must be importable via `__init__.py`.
5. **Pull Request**: Submit a PR with a detailed explanation of the reconnaissance logic and any new dependencies required.

### 🐛 Reporting Vulnerabilities
If you find a security flaw within Chakraview itself, please open an Issue with the prefix `[VULNERABILITY]`.
